Маска для поля с телефоном.

Требования:

  • Пустое поле содержит html-плейсхолдер "Введите телефон". При фокусировке в поле появляется маска 
    +7 (___) ___-__-__
    .
  • Полностью заполненное поле отображает отформатированный вывод: +7 (999) 999-99-99.
  • Неполностью заполненное поле отображает отформатированный вывод введенных символов + остаток маски: 
    +7 (999) 999-9_-__
    .
  • При печати в поле нельзя ввести никакие символы, кроме цифр.
  • Поле реагирует на вставку (подсказка: рассмотрите событие 'input'). Если в тексте вставки есть символы кроме цифр или пробелов, вставка игнорируется. Если только цифры и пробелы — вставляются, стирая предыдущее значение. Исключение — вставки 
    +7 (999) 999-99-99
     и 
    +79999999999
     заполняются корректно.
  • После того, как номер телефона введен полностью, изменяется текст и href в соответсвующем формате у связанной с полем ссылки. id у ссылки может быть любой, нужную ссылку надо найти через атрибут data-related-link в инпуте.
  • Добавьте CSS по своему усмотрению.
